# 📘 Resumo — Aula 05  
## Introdução ao JavaScript (Vanilla JS)

---

## 🎯 Objetivo da Aula

Compreender os fundamentos do desenvolvimento web utilizando **HTML, CSS e JavaScript**, explorando a estrutura de páginas, estilização visual e introdução à programação com foco em interatividade e manipulação de elementos no navegador.

---

## 📚 Conteúdos abordados

### 🌐 HTML, CSS e JavaScript
As três tecnologias fundamentais do desenvolvimento web:

- **HTML:** Define a estrutura da página (conteúdo).
- **CSS:** Define o estilo visual (cores, layout, fontes).
- **JavaScript:** Adiciona interatividade e dinamismo.

---

### 🏗️ HTML (Estrutura da Página)

HTML é uma linguagem de marcação responsável por organizar o conteúdo.

**Principais elementos:**
- `<html>`, `<head>`, `<body>`
- Cabeçalhos (`<h1>` até `<h6>`)
- Parágrafos (`<p>`) e quebras de linha (`<br>`)
- Links (`<a>`)
- Imagens (`<img>`)
- Listas (`<ul>`, `<ol>`, `<li>`)
- Tabelas (`<table>`, `<tr>`, `<td>`)

**HTML5 trouxe:**
- Tags semânticas como `<header>`, `<footer>`, `<section>`, `<article>`, `<nav>`, `<aside>`
- Melhor organização e SEO

---

### 🎥 Mídias em HTML

Permite adicionar conteúdo multimídia diretamente no navegador:

- **Imagens:** `<img>`
- **Vídeos:** `<video>`
- **Áudios:** `<audio>`

---

### 📝 Formulários e Interação

Formulários permitem coletar dados do usuário:

- `<form>`, `<input>`, `<textarea>`
- Envio de dados para servidor ou manipulação via JS

---

### 🎨 CSS (Estilização)

Responsável pelo design e layout da aplicação.

Estrutura:
seletor {
  propriedade: valor;
}

Tipos de uso:
- Inline
- Interno
- Externo (recomendado)

Principais propriedades:
- color, background-color
- font-size, font-family
- margin, padding
- border
- width, height
- display, position

---

### 🧩 Classes e IDs

- class: reutilizável em vários elementos
- id: único na página

---

### ⚡ JavaScript (Interatividade)

JavaScript é uma linguagem executada no navegador (client-side), responsável por tornar páginas dinâmicas.

Principais usos:
- Manipular conteúdo HTML (DOM)
- Validar formulários
- Criar interações e animações
- Comunicação com servidores

---

### 🔤 Variáveis em JavaScript

- var → escopo de função  
- let → escopo de bloco (mais seguro e recomendado)

JavaScript é case sensitive.

---

### 🔀 Condicionais

Permitem executar lógica com base em condições:

- if, else, else if

---

### 🔧 Funções

Funções são blocos reutilizáveis de código.

Exemplos comuns:
- console.log()
- alert(), prompt()
- document.getElementById()
- addEventListener()

---

### ⚙️ Eventos em JavaScript

Eventos permitem responder às ações do usuário:

Exemplos:
- Clique (click)
- Digitação (keydown)
- Mouse (mouseover)
- Envio de formulário (submit)

Fluxo:
1. Evento ocorre  
2. Listener detecta  
3. Função executa  
4. DOM é atualizado  

---

### 🌍 Múltiplas Páginas

Sites podem ser divididos em várias páginas HTML interligadas por navegação.
